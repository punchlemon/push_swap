# push_swap
You have 2 stacks and have to sort the numbers with the stacks.
## Installation
```bash
git clone https://github.com/punchlemon/push_swap
```
```bash
cd push_swap
```
```bash
git submodule init
```
```bash
git submodule update --remote --recursive
```
## Execution
```bash
make ; ./push_swap
```
